# Traffic Penalties

Traffic penalties are consequences applied when drivers break traffic laws.

Common penalties include:

- Fines
- License points
- License suspension
- Vehicle confiscation

## Examples

### Speeding

Driving above the speed limit can result in:

- monetary fines
- license penalty points

### Running a red light

This violation may result in:

- heavy fines
- license suspension

Always respect traffic lights and road signs.
